[명령어]
Ctrl + C : 캔슬
pwd : 현재경로확인  ( ~ 표시된 경로는 기본 사용자 경로)
su : 루트권한
sudo passwd : 비밀번호 세팅
exit : 빠져나가기
ls -al : 리스트 상세
lshw : 하드웨어 정보
lsusb : usb 정보
htop : 프로세서 현황
cd : 디렉토리 이동
